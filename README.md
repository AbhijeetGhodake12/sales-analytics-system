# Sales Analytics System

A complete end-to-end Python application that performs sales data ingestion, cleaning, validation, analytics, API enrichment, and report generation.

This project demonstrates real-world data engineering and analytics workflows, including file handling with encoding issues, data quality validation, interactive filtering, business analytics, external API integration, and automated report generation.

---

## Features

- Reads pipe-delimited sales data files with encoding handling  
- Cleans and validates transactional data  
- Interactive filtering by region and transaction amount  
- Performs multiple business analytics  
- Enriches sales data using DummyJSON Products API  
- Saves enriched data to file  
- Generates a detailed formatted sales report
